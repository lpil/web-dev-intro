An Introduction to Web Dev
==========================

With Ruby & Sinatra


## Development

```sh
# Install the deps
bundle install

# Run the tests
bundle exec rspec

# Run the linter
bundle exec rubocop

# Run the test watcher
bundle exec guard
```


## Adding a new tutorial

Stick it in `source/tutorials/` as a markdown file with the `.html.md`
extension, and add an entry to `data/tutorials.yml`.
